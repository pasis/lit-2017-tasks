Задача 1.
Дано математическое выражение. Вывести числовой результат вычисления этого
выражения.

Требования:
 - Операции + - * /
 - Операция ^ (возведение в степень)
 - Дробные числа
 - Унарные + -
 - Скобки ( expression )
 - Модуль | expression |
 - Правильный приоритет вычисления, учитывая операции и скобки
 - Проверка деления на 0
 - Проверка на валидность скобочной структуры
 - Проверка на валидность мат. выражения
 - Пробелы (white symbols)
 - Функция sqrt() (квадратный корень)

Пример: 2+2*2
        -5+4*3*(2-1)+4/2*(-0.5)+4^0.5

Подзадача 1.
Проверить скобочную структуру, содержащую символы ( и ), на валидность.

Подзадача 2.
Проверить скобочную структуру, содержащую символы ()[], на валидность.

Подзадача 3.
Дана строка символов, удалить из этой строки пробелы.
